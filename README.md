\### Часть 1: Базовый сервис

Описание: сервис на базе FAST API, который обрабатывает HTTP-запросы к БД и возвращает топ N постов по количеству лайков.



\### Часть 2: ML-рекомендательная система

Описание: сервис, который для каждого юзера в любой момент времени возвращает посты, которые пользователю покажут в его ленте соцсети.



Оценка качества модели (метрика Hitrate@5):



$$

\\text{Hitrate}@5 = \\frac{1}{n \\times T} \\times \\sum\_{t=1}^{T} \\left( \\sum\_{i=1}^{n} \\min\\left( 1, \\sum\_{j=1}^{5} \\mathbb\[a\_j(x\_i, t) = 1] \\right) \\right),

$$



где:

 - $n$ - количество юзеров

 - $T$ - количество периодов проверки

 - $a\_j(x\_i, t)$ - $j$-ая рекомендация $i$-ому пользователю в момент времени $t$;
